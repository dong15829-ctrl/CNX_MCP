## 3. 공통 UI·필터

- **기간 선택:** Date Range Picker (from, to)
- **Region / Country:** `getCountry` API → Region 체크박스
- **Site / Local:** `getSiteCode` API → Site/Local 체크박스
- **서브 내비게이션:** Home, Executive Summary, Division Summary, … 등 JavaScript 함수로 이동  
  - 예: `moveMain()`, `moveExecutive()`, `moveDevision()`, `moveSalesAll()`, `moveOperationAll()`, `moveShopappOperation()`, `moveBotjourneyOperation()`, `moveEpp_operation()`, `moveRetailer_operation()`, `moveMxFlagshipSummary()`, `movePerformanceSummary()`
- **Manual & FAQ Download:** `fileDown()` 로 매뉴얼·FAQ 다운로드

## 4. 메뉴별 주요 API (요약)

| 메뉴 | 데이터 API | 차트/그래프 API | 비고 |
|------|------------|-----------------|------|
| Executive Summary | `getExecutiveData` | `getExecutiveGraphData` | 테이블·차트 |

| Operation Summary | `getOperationDataNew`, `getOperationTableData` | `getOperationAllGraphData` | getHybrisType |
| ShopApp Summary | `getShopAppOperationSummaryData`, `getOperationTableData` | `getOperationAllGraphData` | |
| Bot/AI Summary | `getBotJourneyOperationSummaryData`, `getOperationTableData` | `getOperationAllGraphData` | |
| EPP/SMB Summary | `getOperationData`, `getOperationTableData` | `getOperationGraphData` | chat 등 |
| Retailer Summary | `getRetailerOperationData` | `getRetailerOperationGraphData` | |
| MX Flagship Summary | `getFlagshipPreTableData` | — | 테이블 위주 |
| Performance Summary | getPerformanceFunnelData, getPerformanceTrafficTableData, getPerformanceMissedChatTableData, getPerformanceHandledChatTableData | — | 퍼널·테이블 |

공통 파라미터: `title`, `from`, `to`, `period`, `site_code`, `period_date`, `switch_table` 등 (메뉴별 상이).

**정리:**  
globalsess Samsung Global Dashboard는 **로그인 → Main(Home) → 10개 Summary 메뉴** 구조이며, 각 Summary는 **공통 필터(기간·Region·Site)** 와 **데이터/차트 API** 를 사용합니다.  
ePromoter_v1.0에서는 위 문서·설정·분석 JSON을 이용해 동일 기능 재현 및 연동이 가능합니다.

### 데이터 항목·지표 정의

- **지표/컬럼 이름** ✅ 수집 (`indicator_list.json`, `table_headers`).
- **지표 정의** ⚠️ **부분 파악**: `header_tooltips`(툴팁), **API 응답 필드**(`api_responses/*.json`)에서 필드명·구조 추출 반영.  
  매뉴얼·FAQ로 의미·산식·단위 추가 보강 권장.
- **테이블/그래프 → API 파이프라인** ✅ **역참조 명세** 정리:  
  `docs/테이블_그래프_API_역참조_명세.md` 및 `table_api_spec.json`, `graph_api_spec.json` (어느 테이블/그래프가 어느 API를 쓰는지, 요청·응답 스키마 샘플).
