# KODEX 마케팅 인텔리전스: 실행 계획 및 전략

## 1. 프로젝트 개요 (Project Overview)
본 프로젝트는 투자자 심리를 역설계하여 미국 S&P 500 ETF 시장에서 KODEX의 리더십을 탈환하는 것을 목표로 합니다. **100% 웹 크롤링(Web Crawling)**을 통해 텍스트, 오디오, 시각 데이터를 수집하고, AI를 활용해 **7가지 핵심 행동 가설**을 검증하여 데이터 기반의 "행동 마케팅 전략"을 수립합니다.

## 2. 핵심 가설 (Core Hypotheses)
기존 가설을 확장하여 시각적 요소와 알고리즘 요소를 포함한 7가지 가설을 검증합니다.

| ID | 가설 (Hypothesis) | 검증 지표 (Proxy Metric) | 설명 |
|----|-------------------|--------------------------|------|
| **H1** | **검색 점유율** (Search Dominance) | **Share of Search** | 주요 키워드 검색 시 유튜브/네이버 상위 3위를 TIGER가 독점하고 있는가? |
| **H2** | **시각적 유인** (Visual Magnetism) | **Thumbnail Heatmap** | TIGER의 썸네일이 '공포/탐욕' 색상과 텍스트를 사용하여 클릭률(CTR)을 높이는가? |
| **H3** | **언어의 온도** (Tone of Voice) | **Keyword Density** | KODEX는 '상품(이성)'을 설명하고, TIGER는 '수익(감성)'을 이야기하는가? |
| **H4** | **인지 부하** (Cognitive Load) | **Visual Complexity** | KODEX의 상세페이지 도표가 TIGER보다 이해하는 데 2초 더 걸리는가? |
| **H5** | **알고리즘 편향** (Algo-Loop) | **Interaction Velocity** | TIGER 영상이 초반 댓글/좋아요 속도가 빨라 추천 알고리즘의 가산점을 받는가? |
| **H6** | **락인 효과** (Lock-in Effect) | **Co-occurrence** | 커뮤니티에서 TIGER는 '우리(팬덤)'로, KODEX는 '그것(대상)'으로 불리는가? |
| **H7** | **노출 편향** (Exposure Bias) | **Citation Count** | 초보자용 재테크 블로그/카페 글에서 TIGER가 '기본 예시'로 더 많이 인용되는가? |

## 3. 데이터 수집 전략 (Stealth Crawler)
공식 API 없이 **Playwright**와 **Scrapy**를 활용하여 실제 사람처럼 데이터를 수집합니다.

### A. 유튜브 인텔리전스 (YouTube Intelligence)
- **수집 방식**: Playwright를 이용한 무한 스크롤 및 HTML 파싱.
- **수집 데이터**:
  - **메타**: 조회수, 업로드 날짜, 채널명, 구독자 수.
  - **시각**: 고화질 썸네일 이미지 (-> H2 검증용).
  - **콘텐츠**: 제목, 더보기 설명, **전체 자막(CC)** (-> H3 검증용), 댓글(대댓글 포함).

### B. 네이버 인텔리전스 (Naver Intelligence)
- **수집 방식**: 통합검색 및 View 탭(블로그/카페) 크롤링.
- **수집 데이터**:
  - **자동완성**: 검색어 입력 시 추천되는 연관검색어 순서 변화 (-> H1 검증).
  - **블로그**: 상위 노출 포스팅의 본문 텍스트 및 포함된 이미지/도표 (-> H4, H7 검증).
  - **카페**: 주요 재테크 카페의 게시글 및 댓글 타임스탬프.

### C. 구글 및 금융 사이트 (Google & SEO)
- **수집 방식**: SERP(검색 결과 페이지) 분석.
- **수집 데이터**: '미국 ETF 추천' 검색 시 상단 노출되는 티스토리/워드프레스 블로그 내 추천 상품명 추출.

## 4. 비주얼 인텔리전스 (Visual Intelligence)
GPT-4o Vision 또는 Gemini Pro를 활용하여 텍스트 너머의 이미지를 분석합니다.

### A. 썸네일 분석기 (Thumbnail Analyzer)
- **입력**: 유튜브 썸네일 1,000장 (상위 조회수 vs 하위 조회수).
- **분석 로직**:
  - **OCR**: 썸네일 내 텍스트("긴급", "폭락", "지금 사세요") 추출.
  - **객체 인식**: 사람 얼굴(표정), 상승 화살표(Red Arrow), 그래프 유무.
  - **컬러 팔레트**: 브랜드 컬러(파랑/주황) vs 자극적 컬러(빨강/노랑) 비중 분석.

### B. 복잡도 측정기 (Complexity Scorer)
- **입력**: 블로그/상세페이지 내 ETF 구조 설명 도표.
- **분석 로직**:
  - **정보 밀도**: 이미지 내 텍스트와 도형의 개수를 통한 밀도 산출.
  - **가독성 평가**: AI에게 *"이 이미지를 보고 5초 안에 상품 구조를 이해할 수 있습니까?"*라고 질문하여 점수화 (1~10점).

## 5. 실행 전략 (Actionable Execution)
분석 결과는 즉시 실행 가능한 마케팅 가이드로 변환됩니다.

1.  **카피라이팅 생성기 (Brain Agent)**:
    - H3 분석에서 도출된 '트리거 키워드'를 포함한 유튜브 제목/썸네일 문구 자동 생성.
    - *예시*: "S&P500 상품 설명" -> "은퇴 후 월 200만원 만드는 S&P500 투자법".

2.  **트렌드 재킹 알림 (Pipeline Agent)**:
    - 실시간 크롤링 중 '특정 키워드'(예: 트럼프, 금리인하) 급상승 시 알림.
    - 과거 KODEX 영상 중 해당 키워드와 연관된 콘텐츠를 찾아 재업로드/쇼츠화 제안.

3.  **비주얼 A/B 테스트 가이드 (Canvas Agent)**:
    - H2 분석을 통해 도출된 "이기는 디자인 패턴" 가이드라인 배포.
    - *예시*: "파란 배경보다 검은 배경에 노란 텍스트를 썼을 때 클릭률 3.5배 상승".
