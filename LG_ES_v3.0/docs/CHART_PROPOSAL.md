## 1. 제안 차트 ① — Total Score 기준 에러율/개선율 트렌드

### 1.1 목적

> 특정 **기준월(Base Month)** 대비 각 Region의 Total Score가 **얼마나 개선되었는지 / 악화되었는지**를 한눈에 파악

### 1.2 제안 A: 변화율 라인 차트 (Improvement Rate Line Chart) ⭐ 추천

**개념**: 기준월의 Total Score를 0%로 놓고, 이후 각 월의 변화율(%)을 Region별 라인으로 표현

```
  변화율(%)
   +8% ┤                                          ●─── ASIA
       │                                    ●────●
   +6% ┤                              ●────●
       │                         ●───●
   +4% ┤                    ●───●                  ●─── CIS
       │               ●───●               ●─────●
   +2% ┤          ●───●               ●───●
       │     ●───●               ●───●
    0% ┤─●──────────────────────────────────────────── 기준선 (Base Month)
       │     ●───●
   -2% ┤          ●───●                             ●─── EU
       │               ●───●               ●─────●
   -4% ┤                    ●───●      ●───●
       │                         ●───●
   -6% ┤
       └──┬────┬────┬────┬────┬────┬────┬────┬────┬──
         Base  +1   +2   +3   +4   +5   +6   +7   +8  (월)
         2025  2025 2025 2026 2026 ...
          -05  -06  -07  -01  -02
```

**X축**: 시간 (월, 기준월부터 현재까지)  
**Y축**: 기준월 대비 Total Score 변화율 (%)  
**시리즈**: Region별 라인 (Region 고유 색상)  
**기준선**: 0% 수평선 (기준월 = 변화 없음)

**변화율 계산**:
```
변화율(%) = ((현재월 Total Score - 기준월 Total Score) / 기준월 Total Score) × 100
```

**UI 상호작용**:
- 기준월 드롭다운: 사용자가 Base Month를 선택 (기본: 6개월 전)
- 호버 시 툴팁: "ASIA: +5.2% (기준 48.6 → 현재 51.1)"
- 라인 클릭 시 해당 Region 상세 차트로 이동 (제안 ②와 연동)

**기준선(0%)을 중심으로**:
- 위쪽 영역: 개선 (녹색 반투명 배경)
- 아래쪽 영역: 악화 (빨간 반투명 배경)

```
┌─────────────────────────────────────────────────────────────┐
│  Total Score 변화율 트렌드 (기준: 2025-05)                    │
│  [기준월: 2025-05 ▼]                                         │
│                                                             │
│       ┌──────────────────────────────────────────────┐      │
│  +8%  │ ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░ (개선 영역) ░░│      │
│  +6%  │ ░░░░░░░░░░░░░░░░░░░░░░ ●─●─── ASIA ░░░░░░░░│      │
│  +4%  │ ░░░░░░░░░░░░░░░ ●──●─●  ░░░░░░░░░░░░░░░░░░│      │
│  +2%  │ ░░░░░░░ ●──●──●  ░░░ ●─●─●─── Global ░░░░░│      │
│   0%  │─●──●─────────────────────────── 기준선 ──────│      │
│  -2%  │ ▓▓▓ ●──● ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│      │
│  -4%  │ ▓▓▓▓▓▓▓ ●──●──● ▓▓▓▓▓▓▓ ●─●─── EU ▓▓▓▓▓▓│      │
│  -6%  │ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓ ●──●  (악화 영역) ▓▓▓▓▓▓▓▓▓│      │
│       └──┬───┬───┬───┬───┬───┬───┬───┬───┬───┬──┘      │
│        05   06  07  08  09  10  11  12  01  02           │
│        2025                                2026          │
│                                                             │
│  범례: ● ASIA  ● CIS  ● EU  ● Global  ● India             │
│        ● LATAM  ● MEA  ● NA                                │
└─────────────────────────────────────────────────────────────┘
```

**Chart.js 구현 방향**:
```javascript

{
```

### 1.4 제안 C: 스캐터 쿼드런트 차트 (Scatter Quadrant)

**개념**: X축 = 기준월 점수, Y축 = 현재월 점수. 대각선(y=x) 위 = 개선, 아래 = 악화.

```
┌─────────────────────────────────────────────────────────────┐
│  Score 변화 분석 (기준: 2025-10 → 현재: 2026-01)              │
│                                                             │
│  현재월                                                      │
│  60 ┤                               ◆ ASIA                  │
│     │                        ○ Global    ↗ 개선 영역          │
│  55 ┤               ◇ CIS        /                          │
│     │          △ NA          /                               │
│  50 ┤     □ LATAM       /                                    │
│     │  ▽ MEA       / ← y=x (변화 없음)                       │
│  48 ┤           /                                            │
│     │       / ◁ India                                        │
│  45 ┤    /         ↙ 악화 영역                                │
│     │ /      ▷ EU                                            │
│  42 ┤                                                        │
│     └──┬────┬────┬────┬────┬────┬────┬────┬────┬──           │
│       42   45   48   50   52   55   58   60                  │
│                     기준월                                    │
│                                                             │
│  버블 크기 = SKU 수                                           │
│  대각선 위 = 개선, 대각선 아래 = 악화                            │
└─────────────────────────────────────────────────────────────┘
```

**X축**: 기준월 Total Score  
**Y축**: 현재월 Total Score  
**대각선**: y = x (변화 없음 기준)  
**버블 크기**: 해당 Region의 SKU 수  
**색상**: Region 고유 색상

**장점**: 개선/악화 + 현재 수준 + 규모를 한 화면에 표현

## 2. 제안 차트 ② — Region 선택 시 상세 트렌드

#### 레이더 차트 상세

**B2B (5축)**:
```
          Title (20)
            │
   Alt (15) ┼─────── Desc (20)
            │
  Canon (15)┼─── H1 (15)
```

- 각 축은 해당 항목의 만점으로 정규화 (0~100%)
- 3개의 폴리곤 (월별), 최신월이 가장 바깥 = 개선
- 반투명 fill로 겹쳐 보이게

**B2C (10축)**:
```
            UFN (10)
          ╱    ╲
  Alt Front    Basic Assets
       │          │
  Alt Feature  Spec Summary
       │          │
  Canonical      FAQ
          ╲    ╱
   H1 ─── Description
            │
          Title
```

- 10개 축으로 레이더 차트
- 항목 수가 많아 라벨이 겹칠 수 있으므로 약어 사용

**Chart.js 구현**:
```javascript
// Radar Chart
{
  type: 'radar',
  data: {
    labels: ['Title', 'Description', 'H1', 'Canonical', 'Alt Text'],  // B2B
    datasets: [
      {
        label: '2025-11',
        data: [51, 41.5, 45.3, 47.3, 41.3],  // 만점 대비 % 변환
        borderColor: '#C41E3A',
        backgroundColor: 'rgba(196, 30, 58, 0.08)',
        borderWidth: 2,
        pointRadius: 4,
      },
      {
        label: '2025-12',
        data: [52.5, 42.5, 46.7, 46.7, 43.3],
        borderColor: '#6b7280',
        backgroundColor: 'rgba(107, 114, 128, 0.08)',
        borderWidth: 2,
        pointRadius: 4,
      },
      {
        label: '2026-01',
        data: [54, 44, 48, 45.3, 46.7],
        borderColor: '#ea580c',
        backgroundColor: 'rgba(234, 88, 12, 0.12)',
        borderWidth: 2,
        pointRadius: 4,
      },
    ]
  },
  options: {
    scales: {
      r: {
        min: 0,
        max: 100,
        ticks: { stepSize: 20, callback: v => v + '%' },
        pointLabels: { font: { size: 12, weight: 600 } },
      }
    },
    plugins: {
      tooltip: {
        callbacks: {
          label: (ctx) => {
            const maxScores = [20, 20, 15, 15, 15];
            const rawScore = (ctx.parsed.r / 100 * maxScores[ctx.dataIndex]).toFixed(1);
            return `${ctx.dataset.label}: ${rawScore} (${ctx.parsed.r.toFixed(1)}%)`;
          }
        }
      }
    }
  }
}
```

### 2.4 제안 E: 스몰 멀티플 라인 차트 (Small Multiples) — 대안

**개념**: 각 항목별로 작은 라인 차트를 그리드로 배치. 한눈에 모든 항목의 트렌드 방향 파악.

```
┌─────────────────────────────────────────────────────────────────────┐
│  ASIA Region — 항목별 3개월 트렌드                  [Region: ASIA ▼]│
│                                                                     │
│  ┌── Total Score % ──┐  ┌── 1. Title (20) ──┐  ┌── 2. Desc (20) ──┐│
│  │ 51.1              │  │ 10.8              │  │ 8.8              ││
│  │      ╱──●         │  │      ╱──●         │  │      ╱──●        ││
│  │   ╱──            │  │   ╱──            │  │   ╱──           ││
│  │ ●                 │  │ ●                 │  │ ●                ││
│  │ 48.6     ▲ +2.5   │  │ 10.2     ▲ +0.6   │  │ 8.3     ▲ +0.5  ││
│  │ 11  12  01        │  │ 11  12  01        │  │ 11  12  01       ││
│  └───────────────────┘  └───────────────────┘  └──────────────────┘│
│                                                                     │
│  ┌── 3. H1 (15) ────┐  ┌── 4. Canonical(15)┐  ┌── 5. Alt (15) ───┐│
│  │ 7.2               │  │ 6.8               │  │ 7.0              ││
│  │      ╱──●         │  │ ●                 │  │       ╱──●       ││
│  │   ╱──            │  │  ╲──              │  │    ╱──          ││
│  │ ●                 │  │      ╲──●         │  │ ●                ││
│  │ 6.8     ▲ +0.4    │  │ 7.1     ▼ -0.3    │  │ 6.2     ▲ +0.8  ││
│  │ 11  12  01        │  │ 11  12  01        │  │ 11  12  01       ││
│  └───────────────────┘  └───────────────────┘  └──────────────────┘│

```

**장점**:
- 각 항목의 **방향(▲/▼)이 즉시** 눈에 들어옴
- 어떤 항목이 악화되고 있는지 빠르게 파악 (빨강 표시)
- 공간 효율적

**B2C 적용 시**:
- 10개 항목 → 2행 5열 또는 3행 4열 그리드
- 항목명 약어 사용 (UFN, BA, SS, FAQ, Title, Desc, H1, Canon, AF, AFr)
